Welcome to Legal Case Management System, Bangladesh

LCMS is a criminal case management system of People's Republic of Bangladesh. It is a state owned criminal database management system of our country. It holds all the information related to criminal and criminal activities. Its main stakeholders are:

  • Bangladesh Police
  • Bangladesh Courts
  • Bangladesh Jail

System Information

This system was written with Java programming language. It's frontend is designed with typescript programming language with Angular framework. This project was based on Angular 15. It's backend is developed with Spring Boot framework. Spring Boot 2.7 version and maven Framework was used.
